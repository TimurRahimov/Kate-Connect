Нужно создать файловую базу данных, которую можно будет использовать на Selectel.
Преимущества этой БД выражаются в простоте её использования, в простоте настройки
и в том, что все данные будут храниться по папкам и по файлам с расширениями JSON
(или другое).

В Kate Connect эта БД будет встраиваться как модуль. В KC не будет реализации БД:
только использование готового компонента.

Таблицы в данной БД будут представлены в виде папок, а записи - в виде файлов 
с ключевым названием и набором полей. В корне каждой папки (таблицы) будет 
храниться специальный файл с метаданными, разъясняющий, какие данные в нем 
хранятся.

Также в БД должна быть функция создания резервной копии таблицы и самой БД, а
также создания слепка, скриншота БД в виде .zip архива.
